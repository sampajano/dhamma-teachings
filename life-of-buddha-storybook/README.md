# The Life of the Buddha - A Picture Storybook

An illustrated, narrated storybook of the Buddha's life.

Images were originally idealized and generated by May using ChatGPT. 🙏

## Open

Open [`index.html`](index.html) in a browser, or host this repository as a static site.

The page includes:

- 42 illustrated scenes
- English story text
- Per-scene MP3 narration
- Continuous playback with automatic page advancement after the reader clicks Play

## Regenerating Audio

Set `OPENAI_API_KEY`, then run from the repository root:

```bash
python3 life-of-buddha-storybook/scripts/generate-audio.py --concurrency 4
```

Existing audio files are skipped unless `--overwrite` is passed.
