# Dhamma Teachings

Public-facing Dhamma teaching materials, storybooks, and supporting assets.

## Life of the Buddha Storybook

Open the storybook:

- Local/static entry: [`life-of-buddha-storybook/`](life-of-buddha-storybook/)
- Main HTML page: [`life-of-buddha-storybook/index.html`](life-of-buddha-storybook/index.html)
- English Markdown source: [`life-of-buddha-storybook/content/the-life-of-the-buddha-picture-storybook.en.md`](life-of-buddha-storybook/content/the-life-of-the-buddha-picture-storybook.en.md)
- Chinese Markdown source: [`life-of-buddha-storybook/content/the-life-of-the-buddha-picture-storybook.zh.md`](life-of-buddha-storybook/content/the-life-of-the-buddha-picture-storybook.zh.md)

The storybook is a static HTML reader with images, per-page narration, and continuous playback. It can be hosted directly with GitHub Pages, Vercel, Netlify, or any static file server.

## Hosting

GitHub Pages:

1. Push this repository to GitHub.
2. Enable Pages for the `main` branch.
3. Serve from the repository root.

Vercel:

1. Import the repository.
2. Use the default static project settings.
3. No build command is required.
